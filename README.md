# ♟️ Othello (Reversi) AI – Εργασία Τεχνητής Νοημοσύνης

Αυτό το project αποτελεί εργασία για το μάθημα **Τεχνητή Νοημοσύνη** του  
Τμήματος **Πληροφορικής** του **Οικονομικού Πανεπιστημίου Αθηνών (ΟΠΑ)**.

Η εργασία αφορά την υλοποίηση του κλασικού παιχνιδιού **Othello (Reversi)**, με στόχο την ανάπτυξη ενός ισχυρού αντιπάλου τεχνητής νοημοσύνης.

---

## 🧠 Τεχνητή Νοημοσύνη & Στρατηγική

Ο πυρήνας του AI παίκτη βασίζεται στον αλγόριθμο αναζήτησης **Minimax** σε βάθος, ο οποίος βελτιστοποιείται μέσω του αλγορίθμου **Alpha-Beta Pruning**.

### Φασιακές Ευρετικές (Phase-Dependent Heuristics)

Η αξιολόγηση του ταμπλό (evaluate_board) χρησιμοποιεί ένα συνδυασμό διαφορετικών ευρετικών συναρτήσεων, οι οποίες ζυγίζονται δυναμικά ανάλογα με τη **φάση του παιχνιδιού** (Αρχή, Μέσον, Τέλος).

| Φάση Παιχνιδιού | Περιγραφή | Κυρίαρχες Ευρετικές |
| :--- | :--- | :--- |
| **Αρχή** | Έως 10 κομμάτια στο ταμπλό | **Θέση** (Static Weighted) |
| **Μέσον** | Μέχρι τα μέσα του παιχνιδιού | **Θέση** (Position), **Κινητικότητα** (Mobility), **Σύνορο** (Frontier) |
| **Τέλος** | Τελικές κινήσεις | **Διαφορά Κομματιών** (Piece Difference) |

#### Ευρετικές Συναρτήσεις:
* **Weighted Evaluation**: Αξιολογεί τη στρατηγική θέση των κομματιών με βάση προκαθορισμένους πίνακες βαρών (π.χ., οι γωνίες έχουν μεγάλη αξία).
* **Mobility**: Μετράει τη διαφορά στις διαθέσιμες νόμιμες κινήσεις μεταξύ του AI και του αντιπάλου.
* **Frontier Disks**: Μετράει πόσα κομμάτια κάθε παίκτη συνορεύουν με κενό τετράγωνο (ευάλωτα κομμάτια).

## 🚀 Εκτέλεση του Project με Docker Compose

Το project είναι δομημένο ως client-server (Angular Frontend / FastAPI Backend) και μπορεί να εκτελεστεί πλήρως σε ένα απομονωμένο περιβάλλον χρησιμοποιώντας το Docker Compose.

### Προαπαιτούμενα

Βεβαιωθείτε ότι έχετε εγκατεστημένο το **Docker** (το οποίο συνήθως περιλαμβάνει και το Docker Compose):

* [Εγκατάσταση Docker (Docker Desktop)](https://docs.docker.com/desktop/)

### Οδηγίες Εκτέλεσης (Βήμα προς Βήμα)

1.  **Κλωνοποίηση του Repository:**
    Ανοίξτε το Terminal ή το Command Prompt και κατεβάστε τον κώδικα:
    ```bash
    git clone https://github.com/jfotinias/Othello-Reversi-.git
    ```

2.  **Μετάβαση στον Φάκελο:**
    Μεταβείτε στον κύριο φάκελο του project (όπου βρίσκεται το `docker-compose.yml`):
    ```bash
    cd othello
    ```

3.  **Εκτέλεση του Docker Compose:**
    Εκτελέστε την ακόλουθη εντολή. Αυτή η εντολή θα χτίσει τα images (Frontend και Backend) και θα ξεκινήσει τις υπηρεσίες:
    ```bash
    docker compose up --build
    ```
    *Σημείωση: Αν θέλετε οι υπηρεσίες να τρέχουν στο background, προσθέστε την παράμετρο `-d`.*

4.  **Πρόσβαση στην Εφαρμογή:**
    Μόλις δείτε τα μηνύματα επιτυχούς εκκίνησης, ανοίξτε τον browser σας και επισκεφθείτε:
    ```
    http://localhost:4200
    ```
    Η εφαρμογή Othello είναι έτοιμη για χρήση!

5.  **Τερματισμός:**
    Για να σταματήσετε και να καθαρίσετε τους containers όταν ολοκληρώσετε:
    * Πατήστε **Ctrl + C** στο Terminal όπου τρέχει το `docker compose up`.
    * Ή, αν τρέχει στο background, εκτελέστε: `docker compose down`
