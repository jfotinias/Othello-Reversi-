# Εργασία Othello – Τεχνητή Νοημοσύνη

Αυτό το project αποτελεί εργασία για το μάθημα **Τεχνητή Νοημοσύνη** του  
Τμήματος **Πληροφορικής** του **Οικονομικού Πανεπιστημίου Αθηνών (ΟΠΑ)**.

Η εργασία αφορά την υλοποίηση του παιχνιδιού **Othello (Reversi)**, με στόχο
την ανάπτυξη ενός προγράμματος που μπορεί να παίζει στρατηγικά, αξιοποιώντας
τεχνικές αναζήτησης και ευρετικές μεθόδους.

## Περιγραφή
- Ανάπτυξη του παιχνιδιού Othello με χρήση τεχνητής νοημοσύνης.
- Εφαρμογή του αλγορίθμου **Minimax** με πριόνισμα α-β.

## Εκτέλεση
1. Κατέβασε ή κλωνοποίησε το project.
2. Τρέξε το αρχείο `main.py` για να ξεκινήσει το παιχνίδι.
